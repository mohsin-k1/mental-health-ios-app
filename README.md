# Mental Health iOS Application

## Overview

This project was developed as part of my BSc (Hons) Software Engineering degree at Nottingham Trent University.

The application was designed to help users monitor and improve their mental wellbeing by allowing them to securely create an account, record their daily mood, set wellbeing goals and attach photos to their journal entries.

---

## Technologies Used

- Swift
- Xcode
- Firebase Authentication
- Firebase

---

## Features

- User registration
- Secure user login
- Mood tracking
- Goal setting
- Camera integration
- Personal journal entries
- Secure cloud data storage using Firebase

---

## Application Functionality

Users can:

- Create an account
- Log into the application securely
- Record their daily mood
- Track previous mood entries
- Create wellbeing goals
- Capture photos using the device camera
- Store and retrieve personal data using Firebase
